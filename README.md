## Senior Backend Engineer | Java | Spring Boot | Quarkus

Desenvolvedor Backend Java com atuação em sistemas críticos de alta disponibilidade e grande volume transacional, atualmente envolvido em soluções do setor financeiro para a Caixa Econômica Federal.

Possuo experiência no desenvolvimento de APIs REST e microsserviços utilizando Java, Spring Boot e Quarkus, além de atuação com mensageria, arquiteturas distribuídas, integração entre serviços e pipelines CI/CD.

Minha trajetória profissional combina desenvolvimento backend com mais de 16 anos de experiência prévia em Infraestrutura de TI e liderança técnica, proporcionando uma visão ampla sobre ambientes produtivos, observabilidade, monitoramento, troubleshooting e resposta a incidentes em produção.
